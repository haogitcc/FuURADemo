Troubleshoot:

Error: No response from reader at any baud rate

  • Make sure that device is powered up and connected.
  • Make sure that serial cable is plugged at module side and at host side.

Error: Application image failed CRC check

  • Re-install the firmware on module using any other tool i.e Universal Reader Assistant or sample MercuryAPI firmware.cs or .java or .c codelet and then try to reconnect.

Error: Invalid argument

  • Refresh the port list and then try to reconnect again.

Error: Connection Timed Out

  • If the physical connection is broken or module power goes off while read is in progress

Problem: No ports displayed under available devices

  • Make sure device is powered on and connected.
  • Refresh the port list by clicking the refresh button (curved arrows icon) which is next to "Available Devices" text in connect tab.

Problem: No tags displayed on the Read tab

  • Make sure an antenna was connected to the module, and to the port that was configured to read on in the Configure tab, and there are tags in the field.
