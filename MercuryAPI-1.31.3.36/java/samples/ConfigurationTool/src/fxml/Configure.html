Configure:

Read & Write Settings and Performance Tuning

  • These settings define the persistent operational configuration that will be saved to the module.

Display Options

  • Defines the additional metadata fields that will be returned with each tag read and displayed on when using the Read tab.

Profile

  • Allows the settings selected in Read & Write, Performance Tuning and Display Options to be saved to a file. The saved options can then be loaded when connected to another module to simplify applying common settings to a number of modules.

Firmware Update

  • Install a new firmware image on the module. NOTE: This will reset autonomous configuration settings.

Apply Button

  • When clicked all the settings specified in the other sections will be persistently stored on the module. This will cause the module to boot into autonomous mode.
